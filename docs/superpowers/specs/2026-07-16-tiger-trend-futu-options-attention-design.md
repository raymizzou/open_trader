# 老虎美股趋势与富途跨市场期权关注设计

## 状态

- 日期：2026-07-16
- 状态：已确认
- 原型：统一列表版已确认方向，尚未进入生产实现

## 决策摘要

- 老虎改为美股趋势交易账户，取代既有 `tiger_sma200_equal_weight/v1`。
- 辉立继续作为港股趋势交易账户，东方财富继续作为 A 股趋势交易账户。
- 富途作为期权增强账户，聚合老虎美股和辉立港股的趋势变化；不独立计算第三套趋势。
- 富途只提示值得关注的底层标的，不推荐期权合约、到期日、行权价或自动下单。
- Dashboard 使用固定列顺序的统一列表，不生成 LLM 式标题或摘要。

本设计替代旧文档中“富途是美股趋势账户”和“老虎是 SMA200 长期账户”的账户定位。旧文档保留为历史决策记录，不原地改写。

## 账户与入口

| 账户 | Dashboard 入口 | 交易报告 | 期权关注来源 |
| --- | --- | --- | --- |
| 老虎 | 美股趋势报告 | 美股 | 同一份美股趋势变化 |
| 辉立 | 港股趋势报告 | 港股 | 同一份港股趋势变化 |
| 东方财富 | A 股趋势报告 | A 股 | 不进入富途期权关注 |
| 富途 | 期权关注 | 无独立交易报告 | 老虎美股 + 辉立港股 |

富途自选股不限定关注范围。关注范围只来自老虎美股与辉立港股趋势报告实际覆盖的持仓和 Trend Animals 池。

## 数据职责

- Trend Animals：US、HK、CN 三个市场统一请求趋势字段。
- 老虎账户：提供美股持仓、净值和现金快照。
- 辉立：提供港股账户和日结单数据。
- 东方财富：提供 A 股账户数据。
- Futu OpenD：继续提供 US/HK 行情和日 K；只在 Trend Animals 扩展字段缺失时补充既有日线指标，不改变老虎或辉立的交易规则。

数据流只有两份可进入富途的源报告：

```text
Trend Animals + Tiger account + Futu quotes/K-line
                    -> trend_us_tiger -> Tiger US report
                                      -> Futu US option attention

Trend Animals + Phillips account + Futu quotes/K-line
                    -> trend_hk_phillips -> Phillips HK report
                                           -> Futu HK option attention
```

富途期权关注是两份源报告的只读投影，不保存或计算一份独立趋势结论。

## 统一字段

三个市场都请求同一组 Trend Animals 字段：

- 身份：代码、名称、行业、可交易状态。
- 市场：价格、市值、成交额。
- 趋势：右侧状态、右侧天数、右侧累计涨幅、温度、节气。
- 强度：本地强度、全球强度、上周强度、上月强度、强度变化方向。
- 风险：危险、沸腾、开香槟。
- 标签：Trend Animals 返回的原始标签。

核心字段中的右侧状态、危险状态和账户数据必须有效；否则标的或报告进入人工复核。扩展字段缺失不阻断报告，必须在固定位置显示 `未提供`，不得移动其他字段补位。

按当前 50 个标的估算，统一字段请求约为每日 `3.55` 个 API 计费单位；按最近实际调用基线估算约为每日 `3.97` 个单位、21 个交易日约 `83.4` 个单位。Trend Animals API 未声明该单位对应的货币。

## 趋势变化

富途只显示源报告识别出的字段变化，不由 LLM 编写解释性标题。以下事件可以使标的进入关注列表：

- 右侧状态变化。
- 趋势温度变化。
- 趋势节气变化。
- 危险、沸腾或开香槟状态变化。
- 周强度方向变化。
- 标的首次进入有效趋势范围。

活动保护线首次触发不属于趋势变化，不进入富途期权关注。

首次迁移时，可用 2026-07-15 富途美股趋势报告为重合标的建立一次基线；没有基线的标的以“新进入关注范围”处理，不伪造前值。

## Dashboard 列表

富途入口打开一个统一列表，美股和港股只用分组行区隔。所有标的固定使用以下列顺序：

1. 标的。
2. 分类。
3. 右侧状态。
4. 趋势温度。
5. 趋势节气。
6. 本地 / 全球强度。
7. 上周 / 上月强度。
8. 右侧天数 / 累计涨幅。
9. 危险 / 沸腾 / 开香槟。
10. 来源动作。

列表不显示“首次进入关注范围”“危险信号首次出现”等合成句子。发生变化的原始字段本身高亮；没有变化的字段保持普通样式。

桌面端使用表格。移动端允许每行折成带固定字段标签的两列或一列，但字段顺序不变，不产生横向滚动。

## 更新与过期状态

美股和港股分别维护状态，互不连带：

1. **当天数据已更新**：展示最新快照，允许与前一份有效快照比较并生成趋势变化。
2. **当天数据尚未更新，但存在旧快照**：继续展示旧列表，并明确标注 `数据截至 {日期}；今日未更新`；不生成新的趋势变化，也不新增期权关注。
3. **没有任何有效快照**：该市场显示 `暂时不可用`。
4. **稍后更新成功**：页面直接切换到最新快照，恢复变化判断；不在页面保留重复版本。

“直接切换”只描述 Dashboard 展示。源市场的冻结日报仍按既有审计规则保留；富途不复制日报历史，只读取源报告的最新和前一份有效快照。

账户新鲜度与趋势源新鲜度分开处理：

- Trend Animals 尚未更新：沿用上面的旧列表规则，不宣称新变化。
- 老虎账户快照有效但非当天：可展示趋势，但禁止新增买入建议，现有持仓进入人工复核。
- 老虎账户没有有效快照：美股趋势报告生成失败。

## 老虎仓位与执行

- 新仓目标统一按老虎完整账户净值的 `4%` 计算。
- 计算基准统一换算为 HKD，USD/HKD 固定使用 `7.85`。
- 买入总额不得超过所有币种净现金换算后的可用金额，不使用融资。
- 最多持有 10 个普通美股或 ETF 标的。
- 所有动作只作人工执行建议，不自动下单。

美股保护线继续存在于老虎趋势报告和 watcher 中，行情仍可由 Futu OpenD 提供；通知身份必须标为老虎。保护线触发不进入富途的趋势变化列表。

## 飞书

- 老虎每天发送一条美股趋势报告，其中附美股期权关注。
- 辉立每天发送一条港股趋势报告，其中附港股期权关注。
- 东方财富维持独立 A 股趋势报告，不附期权关注。
- 富途不再额外发送第三条汇总消息。
- 无动作和失败通知继续按各源市场的每日账本去重。

- 迁移完成后删除旧 `trend_us_futu` 运行状态和报告产物；需要的 2026-07-15 基线先一次性提取。

## 验收

- 老虎、辉立、东方财富和富途四个入口符合新的账户定位。
- 富途列表字段顺序固定，美股和港股一致，缺失字段不换位。
- 页面没有 LLM 合成趋势标题。
- 美股未更新、有旧快照、无快照和稍后恢复四种状态均正确。
- 更新恢复后只展示最新数据，趋势差异来自相邻有效快照。
- 富途没有独立趋势计算或第三条飞书通知。
- 老虎新仓使用 4% 净值、HKD 口径、7.85 汇率、现金上限和 10 个持仓上限。
- 保护线仍能服务老虎交易报告，但不进入期权关注。
- US/HK/CN 统一请求字段，扩展字段缺失时明确显示 `未提供`。
- 最终实现必须运行真实工作流、检查后台进程和新日志，并以 `make acceptance` 的 `PASS` 作为交付门槛。

## 非目标

- 推荐具体期权合约。
- 自动下单。
- 用富途自选股扩展标的池。
- 把 A 股纳入富途期权关注。
- 新建通用多券商策略框架。
